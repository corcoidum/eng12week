# 12주 적응형 영어 회화 학습 시스템

이 문서는 `Eng12Week` 앱의 학습 설계 요약입니다. 앱은 하루 30분 학습, 진단 기반 트랙 조정, SRS 복습, 주간 평가, AI 대화 프롬프트 생성을 한 화면에서 운영하도록 구현되었습니다.

## 대상과 목표

기초 문법은 알지만 듣기와 말하기가 막히는 한국어 원어민 학습자를 대상으로 합니다. 목표는 원어민 수준의 완벽함이 아니라 익숙한 상황에서 5-10분 동안 대화를 유지하는 실용 회화 능력입니다.

## 핵심 원칙

- 짧고 자주 말하기: 하루 30분 안에 듣기, 따라 말하기, 즉답, 회상, 회고를 모두 포함합니다.
- 진단 기반 조정: 7개 능력 점수를 바탕으로 Speaking, Listening, Balanced 트랙을 추천합니다.
- 회상 중심 복습: 새 표현은 SRS 카드로 저장하고 `1일 -> 3일 -> 7일 -> 14일` 간격으로 복습합니다.
- 안전한 AI 사용: 실제 개인정보, 진료 정보, 환자 정보는 넣지 않고 synthetic scenario만 사용합니다.

## 초기 진단

앱의 Diagnostic 탭은 다음 7개 항목을 1-5점으로 기록합니다.

| 항목 | 의미 |
| --- | --- |
| Speaking Fluency | 멈춤 없이 말하는 정도 |
| Listening Comprehension | 짧은 발화를 이해하는 정도 |
| Vocabulary Recall | 필요한 표현을 바로 떠올리는 정도 |
| Grammar Accuracy | 기본 문장 구조와 시제 정확도 |
| Pronunciation | 발음과 리듬의 전달력 |
| Response Speed | 질문에 바로 반응하는 속도 |
| Conversation Maintenance | 되묻기, 이어 말하기, 마무리 능력 |

추천 트랙 규칙은 단순하게 시작합니다.

- Speaking 점수가 낮으면 Speaking Track
- Listening 점수가 낮으면 Listening Track
- 둘 다 비슷하면 Balanced Track

## 하루 30분 루틴

| 시간 | 활동 | 목적 |
| --- | --- | --- |
| 5분 | 듣기 shadowing | 소리와 리듬 입력 |
| 7분 | 핵심 표현 3개 말하기 | 표현 자동화 |
| 8분 | 즉답 drill | 반응 속도 개선 |
| 7분 | AI role-play | 실제 대화 유지 |
| 3분 | 회고와 SRS 저장 | 다음 복습 연결 |

앱의 Today 탭은 이 루틴을 체크리스트로 운영하며 완료율을 보여줍니다.

## SRS 복습 규칙

카드는 `front`, `back`, `situation`, `dueDate`, `stage`를 가집니다.

- 성공: 다음 stage로 이동하고 복습 간격을 늘립니다.
- 재학습: stage를 초기화하고 다음 날 다시 복습합니다.
- due 카드 수는 상단 배지로 표시합니다.

## 주간 평가와 난이도 조정

Weekly 탭은 fluency, comprehension, confidence 점수를 입력받아 평균을 계산합니다.

- 평균 4.1 이상: 난이도 상승
- 평균 2.5 이하: 난이도 하향
- 그 외: 현재 난이도 유지

이 규칙은 학습자가 과하게 지치지 않도록 단순하고 보수적으로 설계했습니다.

## AI 프롬프트 운영

Prompts 탭은 학습 주제, 난이도, 상황을 조합해 conversation prompt와 feedback prompt를 생성합니다.

권장 방식:

1. conversation prompt로 5-10분 role-play를 시작합니다.
2. 대화 중 막힌 표현을 SRS 카드에 저장합니다.
3. feedback prompt로 transcript를 점검합니다.
4. 다음 날 루틴에서 약한 항목을 집중합니다.

## 개인정보와 의료 안전

의료기관 행정 상황을 연습하더라도 실제 이름, 전화번호, 주민등록번호, 차트번호, 검사결과, 진단명 등은 입력하지 않습니다. 앱과 문서는 synthetic sample과 비민감 workflow 설명만 전제로 합니다.

## 구현 요약

- 정적 웹앱: `index.html`, `styles.css`, `app.js`
- 저장 방식: browser `localStorage`
- 배포 방식: GitHub Pages 또는 정적 호스팅에 바로 업로드 가능
- 외부 dependency: 없음

## 수동 검증 체크리스트

- `index.html`을 브라우저에서 열었을 때 좌측 12주 Roadmap과 상단 진행률이 표시됩니다.
- Today 탭에서 루틴을 체크하면 완료율이 갱신됩니다.
- Diagnostic 탭에서 슬라이더를 움직이면 평균 점수와 추천 트랙이 바뀝니다.
- SRS 탭에서 카드를 추가하고 성공/재학습 상태를 변경할 수 있습니다.
- Weekly 탭에서 점수를 바꾸면 난이도 권장 결과가 바뀝니다.
- Prompts 탭에서 프롬프트를 생성하고 복사할 수 있습니다.
